## 1. Billing Overview

Billing is split across two modules:

| Module | Responsibility |
|--------|---------------|
| **Billing** | Paystack integration, subscriptions, invoices, payments, webhooks |
| **Registration** | New tenant signup flow that orchestrates billing + provisioning |

Both modules operate against `CoreDbContext`. The billing entities (Plan, Subscription, Invoice, Payment) are defined in [02 — Database & Multi-Tenancy](02-database-multitenancy.md) §2.

### Payment Gateway: Paystack

[Paystack](https://paystack.com) is the **only** payment gateway. No Stripe, no alternatives. Paystack is Africa-focused, supports ZAR (South African Rand), and provides:

- Recurring subscriptions
- Plan management
- Transaction verification
- Webhook notifications
- Customer management

> **Note**: Full Paystack API docs will be provided when implementation begins. This document defines the architecture and integration points.

```csharp
    /// </summary>

    /// Process a webhook event from the payment gateway.
    /// </summary>
```

```csharp
public class PaystackClient
{
    private readonly HttpClient _httpClient;
    private readonly PaystackOptions _options;
    private readonly ILogger<PaystackClient> _logger;

    public PaystackClient(HttpClient httpClient, IOptions<PaystackOptions> options,
        ILogger<PaystackClient> logger)
    {
        _httpClient = httpClient;
        _options = options.Value;
        _logger = logger;

        _httpClient.BaseAddress = new Uri("https://api.paystack.co/");
        _httpClient.DefaultRequestHeaders.Authorization =
            new AuthenticationHeaderValue("Bearer", _options.SecretKey);
    }

    // --- Plans ---

    public async Task<PaystackPlanResponse?> CreatePlanAsync(PaystackCreatePlanRequest request)
    {
        var response = await _httpClient.PostAsJsonAsync("plan", request);
        response.EnsureSuccessStatusCode();
        var result = await response.Content.ReadFromJsonAsync<PaystackApiResponse<PaystackPlanResponse>>();
        return result?.Data;
    }

    public async Task<List<PaystackPlanResponse>> ListPlansAsync()
    {
        var response = await _httpClient.GetAsync("plan");
        response.EnsureSuccessStatusCode();
        var result = await response.Content.ReadFromJsonAsync<PaystackApiListResponse<PaystackPlanResponse>>();
        return result?.Data ?? [];
    }

    // --- Transactions ---

    public async Task<PaystackInitializeResponse?> InitializeTransactionAsync(
        PaystackInitializeRequest request)
    {
        var response = await _httpClient.PostAsJsonAsync("transaction/initialize", request);
        response.EnsureSuccessStatusCode();
        var result = await response.Content
            .ReadFromJsonAsync<PaystackApiResponse<PaystackInitializeResponse>>();
        return result?.Data;
    }

    public async Task<PaystackTransactionResponse?> VerifyTransactionAsync(string reference)
    {
        var response = await _httpClient.GetAsync($"transaction/verify/{reference}");
        response.EnsureSuccessStatusCode();
        var result = await response.Content
            .ReadFromJsonAsync<PaystackApiResponse<PaystackTransactionResponse>>();
        return result?.Data;
    }

    // --- Subscriptions ---

    public async Task<PaystackSubscriptionResponse?> CreateSubscriptionAsync(

    {
        var response = await _httpClient.PostAsJsonAsync("subscription", request);
        response.EnsureSuccessStatusCode();
```

### PaystackBillingService

```csharp
public class PaystackBillingService : IBillingService
{
    private readonly PaystackClient _paystack;
    private readonly CoreDbContext _coreDb;
    private readonly ILogger<PaystackBillingService> _logger;
    private readonly IAuditWriter _audit;

    public async Task<SubscriptionInitResult> InitializeSubscriptionAsync(
        SubscriptionInitRequest request)
    {
        // 1. Get the plan
        var plan = await _coreDb.Plans.FindAsync(request.PlanId);
        if (plan is null)
            return new SubscriptionInitResult(false, Error: "Plan not found");

        // 2. Handle free plans — no payment needed
        if (plan.MonthlyPrice == 0)
        {
            await CreateFreeSubscription(request.TenantId, plan);
            return new SubscriptionInitResult(true);
        }

        // 3. Calculate amount (Paystack uses kobo/cents — multiply by 100)
        var amount = request.BillingCycle == BillingCycle.Annual && plan.AnnualPrice.HasValue
            ? plan.AnnualPrice.Value * 100
            : plan.MonthlyPrice * 100;

        // 4. Initialize Paystack transaction
        var paystackResult = await _paystack.InitializeTransactionAsync(new PaystackInitializeRequest
        {
            Email = request.Email,
            Amount = (int)amount,
            Currency = plan.Currency ?? "ZAR",
            CallbackUrl = request.CallbackUrl ?? $"{_options.CallbackBaseUrl}/register/verify",
            Metadata = new Dictionary<string, object>
            {
                ["tenant_id"] = request.TenantId.ToString(),
                ["plan_id"] = request.PlanId.ToString(),
                ["billing_cycle"] = request.BillingCycle.ToString()
            },
            Plan = plan.PaystackPlanCode // Links to Paystack's recurring plan
        });

        if (paystackResult is null)
            return new SubscriptionInitResult(false, Error: "Payment initialization failed");

        // 5. Create pending subscription record
        var subscription = new Subscription
        {
            TenantId = request.TenantId,
            PlanId = request.PlanId,
            Status = SubscriptionStatus.Active, // Will be updated by webhook
            BillingCycle = request.BillingCycle,
            StartDate = DateTime.UtcNow,
            PaystackSubscriptionCode = paystackResult.Reference
        };
        _coreDb.Subscriptions.Add(subscription);
        await _coreDb.SaveChangesAsync();

        return new SubscriptionInitResult(true, PaymentUrl: paystackResult.AuthorizationUrl);
    }

    public async Task<WebhookResult> ProcessWebhookAsync(string payload, string signature)
    {
        // 1. Verify webhook signature
        if (!VerifySignature(payload, signature))
            return new WebhookResult(false, Error: "Invalid signature");

        // 2. Parse event
        var webhookEvent = JsonSerializer.Deserialize<PaystackWebhookEvent>(payload);
        if (webhookEvent is null)
            return new WebhookResult(false, Error: "Invalid payload");

        // 3. Process based on event type
        return webhookEvent.Event switch
        {
            "charge.success" => await HandleChargeSuccess(webhookEvent),
            "subscription.create" => await HandleSubscriptionCreated(webhookEvent),
            "subscription.not_renew" => await HandleSubscriptionNotRenew(webhookEvent),
            "subscription.disable" => await HandleSubscriptionDisabled(webhookEvent),
            "invoice.create" => await HandleInvoiceCreated(webhookEvent),
            "invoice.payment_failed" => await HandlePaymentFailed(webhookEvent),
            _ => new WebhookResult(true) // Acknowledge unhandled events
        };
    }

    private bool VerifySignature(string payload, string signature)
    {
        var hash = HMACSHA512.HashData(
            Encoding.UTF8.GetBytes(_options.WebhookSecret),
            Encoding.UTF8.GetBytes(payload)
        );
        var computedSignature = Convert.ToHexStringLower(hash);
        return CryptographicOperations.FixedTimeEquals(
            Encoding.UTF8.GetBytes(computedSignature),
            Encoding.UTF8.GetBytes(signature)
        );
    }

    // ... webhook handler implementations below
}
```

## 5. Webhook Event Handlers

### Webhook Endpoint

```csharp
[ApiController]
[Route("api/webhooks")]
public class PaystackWebhookController : ControllerBase
{
    private readonly IBillingService _billing;
    private readonly ILogger<PaystackWebhookController> _logger;

    [HttpPost("paystack")]
    public async Task<IActionResult> HandleWebhook()
    {
        // 1. Read raw body
        using var reader = new StreamReader(Request.Body);
        var payload = await reader.ReadToEndAsync();

        // 2. Get signature header
        var signature = Request.Headers["x-paystack-signature"].FirstOrDefault();
        if (string.IsNullOrEmpty(signature))
            return BadRequest("Missing signature");

        // 3. Process
        var result = await _billing.ProcessWebhookAsync(payload, signature);

        // 4. Always return 200 to acknowledge receipt (Paystack retries on non-200)
        return Ok();
    }
}
```

### Handled Webhook Events

| Event | Action |
|-------|--------|
| `charge.success` | Create `Payment` record, update `Invoice` status to Paid, update `Subscription.NextBillingDate` |
| `subscription.create` | Update `Subscription.PaystackSubscriptionCode`, set status to Active |
| `subscription.not_renew` | Set `Subscription.Status` to Cancelled, set `EndDate` |
| `subscription.disable` | Set `Subscription.Status` to Cancelled, suspend tenant if past grace period |
| `invoice.create` | Create `Invoice` record |
| `invoice.payment_failed` | Update `Invoice` status to Overdue, set `Subscription.Status` to PastDue |

### charge.success Handler

```csharp
private async Task<WebhookResult> HandleChargeSuccess(PaystackWebhookEvent webhookEvent)
{
    var data = webhookEvent.Data;
    var reference = data.Reference;

    // Idempotency check — don't process the same transaction twice
    if (await _coreDb.Payments.AnyAsync(p => p.PaystackReference == reference))
        return new WebhookResult(true); // Already processed

    // Extract metadata
    var tenantId = Guid.Parse(data.Metadata["tenant_id"].ToString()!);

    // Create payment record
    var payment = new Payment
    {
        TenantId = tenantId,
        Amount = data.Amount / 100m, // Convert from kobo/cents
        Currency = data.Currency?.ToUpperInvariant() ?? "ZAR",
        Status = PaymentStatus.Success,
        PaystackReference = reference,
        PaystackTransactionId = data.Id?.ToString(),
        GatewayResponse = data.GatewayResponse,
        TransactionDate = DateTime.UtcNow
    };
    _coreDb.Payments.Add(payment);

    // Update related invoice if exists
    var invoice = await _coreDb.Invoices
        .FirstOrDefaultAsync(i => i.PaystackReference == reference);
    if (invoice is not null)
    {
        invoice.Status = InvoiceStatus.Paid;
        invoice.PaidDate = DateTime.UtcNow;
        payment.InvoiceId = invoice.Id;
    }

    // Ensure tenant is active
    var tenant = await _coreDb.Tenants.FindAsync(tenantId);
    if (tenant is not null && tenant.Status == TenantStatus.Suspended)
    {
        tenant.Status = TenantStatus.Active;
    }

    await _coreDb.SaveChangesAsync();

    // Write audit
    await _audit.WriteAsync(new AuditEntry
    {
        TenantSlug = tenant?.Slug,
        EntityType = "Payment",
        EntityId = payment.Id.ToString(),
        Action = "Created",
        Timestamp = DateTime.UtcNow
    });

    return new WebhookResult(true);
}
```

```csharp
private async Task<WebhookResult> HandlePaymentFailed(PaystackWebhookEvent webhookEvent)
{
    var data = webhookEvent.Data;
    var subscriptionCode = data.Subscription?.Code;

    if (subscriptionCode is null)

    var subscription = await _coreDb.Subscriptions
        .Include(s => s.Tenant)
        .FirstOrDefaultAsync(s => s.PaystackSubscriptionCode == subscriptionCode);

    if (subscription is null)
        return new WebhookResult(true);

    subscription.Status = SubscriptionStatus.PastDue;
    await _coreDb.SaveChangesAsync();

    // TODO: After X days of PastDue, suspend the tenant

    return new WebhookResult(true);
```

## 6. Registration Flow — End to End

The complete flow from "user clicks Register" to "tenant is operational":

```
┌──────────────────────────────────────────────────────────────┐
│                    REGISTRATION FLOW                          │
└──────────────────────────────────────────────────────────────┘

Step 1: GET /register
  ├── Show registration form
  ├── Fields: Organisation Name, Admin Email, Desired Slug
  ├── Plan selector (loaded from CoreDbContext.Plans)
  ├── Billing cycle toggle (Monthly / Annual)
  └── Turnstile bot protection widget

Step 2: POST /register (validate)
  ├── Validate Turnstile token
  ├── Validate slug uniqueness (HTMX inline: GET /register/check-slug?slug=xxx)
  ├── Validate email format
  └── If invalid → return form with validation errors (Swap.Htmx partial)

Step 3: POST /register (process)
  ├── Create Tenant in CoreDbContext (status = PendingSetup)
  ├── If FREE plan:
  │     ├── Create free Subscription record
  │     ├── Provision tenant database (see Step 5)
  │     └── Skip to Step 6
  └── If PAID plan:
        ├── Initialize Paystack transaction
        ├── Store pending Subscription record
        └── Redirect to Paystack payment page (Step 4)

Step 4: Paystack Payment (external)
  ├── User completes payment on Paystack's hosted page
  ├── Paystack redirects to callback URL: /register/callback?reference=xxx
  └── Paystack fires webhook: charge.success

Step 5: Tenant Database Provisioning
  ├── Create SQLite file: data/tenants/{slug}.db
  ├── Apply all TenantDbContext migrations
  ├── Set PRAGMA journal_mode=WAL
  ├── Seed default roles (Admin, Member)
  ├── Seed default permissions
  ├── Create admin user (email from registration)
  └── Update tenant status to Active

Step 6: Post-Registration
  ├── Send welcome email with magic link to tenant login
  ├── Redirect to success page: /register/success
  └── Success page shows: tenant URL, "Check your email" message
```

### Registration Controller

```csharp
public class RegistrationController : SwapController
{
    private readonly CoreDbContext _coreDb;
    private readonly IBillingService _billing;
    private readonly ITenantProvisioner _provisioner;
    private readonly IBotProtection _botProtection;
    private readonly IEmailService _email;

    // GET /register
    public async Task<IActionResult> Index()
    {
        var plans = await _coreDb.Plans
            .Where(p => p.IsActive)
            .OrderBy(p => p.SortOrder)
            .ToListAsync();

        return SwapView(new RegisterViewModel { Plans = plans });
    }

    // GET /register/check-slug?slug=xxx (HTMX inline validation)
    public async Task<IActionResult> CheckSlug(string slug)
    {
        slug = slug.ToLowerInvariant().Trim();

        if (slug.Length < 3)
            return Content("<span class='text-error text-sm'>Minimum 3 characters</span>");

        if (!Regex.IsMatch(slug, @"^[a-z0-9][a-z0-9-]*[a-z0-9]$"))
            return Content("<span class='text-error text-sm'>Only lowercase letters, numbers, and hyphens</span>");

        if (ReservedSlugs.IsReserved(slug))
            return Content("<span class='text-error text-sm'>This name is reserved</span>");

        var exists = await _coreDb.Tenants.AnyAsync(t => t.Slug == slug);
        return exists
            ? Content("<span class='text-error text-sm'>Already taken</span>")
            : Content("<span class='text-success text-sm'>Available ✓</span>");
    }

    // POST /register
    [ValidateAntiForgeryToken]
    public async Task<IActionResult> Register(RegisterRequest request)
    {
        // 1. Bot protection
        if (!await _botProtection.ValidateAsync(request.TurnstileToken))
            return SwapView("Index").WithToast("Verification failed. Please try again.", ToastType.Error);

        // 2. Validate
        if (!ModelState.IsValid)
            return SwapView("Index", new RegisterViewModel { Request = request });

        // 3. Check slug availability (race condition guard)
        var slug = request.Slug.ToLowerInvariant().Trim();
        if (await _coreDb.Tenants.AnyAsync(t => t.Slug == slug))
        {
            ModelState.AddModelError("Slug", "This URL is no longer available");
            return SwapView("Index", new RegisterViewModel { Request = request });
        }

        // 4. Get plan
        var plan = await _coreDb.Plans.FindAsync(request.PlanId);
        if (plan is null || !plan.IsActive)
            return SwapView("Index").WithToast("Invalid plan selected", ToastType.Error);

        // 5. Create tenant record
        var tenant = new Tenant
        {
            Name = request.OrganisationName,
            Slug = slug,
            ContactEmail = request.Email,
            PlanId = plan.Id,
            Status = TenantStatus.PendingSetup,
            DatabaseName = $"{slug}.db"
        };
        _coreDb.Tenants.Add(tenant);
        await _coreDb.SaveChangesAsync();

        // 6. Free plan → provision immediately
        if (plan.MonthlyPrice == 0)
        {
            var provisionResult = await _provisioner.ProvisionAsync(new TenantProvisionRequest(
                tenant.Name, tenant.Slug, request.Email, plan.Id
            ));

            if (!provisionResult.Success)
                return SwapView("Index").WithToast("Setup failed. Please try again.", ToastType.Error);

            // Create free subscription
            _coreDb.Subscriptions.Add(new Subscription
            {
                TenantId = tenant.Id,
                PlanId = plan.Id,
                Status = SubscriptionStatus.Active,
                BillingCycle = BillingCycle.Monthly,
                StartDate = DateTime.UtcNow
            });
            await _coreDb.SaveChangesAsync();

            // Send welcome magic link
            await _email.SendMagicLinkAsync(request.Email,
                $"Welcome! Click to access your dashboard.");

            return SwapView("Success", new RegisterSuccessViewModel
            {
                TenantSlug = slug,
                TenantName = tenant.Name
            });
        }

        // 7. Paid plan → initialize payment
        var billingResult = await _billing.InitializeSubscriptionAsync(new SubscriptionInitRequest(
            TenantId: tenant.Id,
            Email: request.Email,
            PlanId: plan.Id,
            BillingCycle: request.BillingCycle,
            CallbackUrl: $"{Request.Scheme}://{Request.Host}/register/callback"
        ));

        if (!billingResult.Success || billingResult.PaymentUrl is null)
        {
            return SwapView("Index")
                .WithToast("Payment setup failed. Please try again.", ToastType.Error);
        }

        // 8. Redirect to Paystack
        return Redirect(billingResult.PaymentUrl);
    }

    // GET /register/callback?reference=xxx (Paystack redirects here after payment)
    public async Task<IActionResult> Callback(string reference)
    {
        // 1. Find the subscription by Paystack reference
        var subscription = await _coreDb.Subscriptions
            .Include(s => s.Tenant)
            .FirstOrDefaultAsync(s => s.PaystackSubscriptionCode == reference);

        if (subscription is null)
            return SwapView("Error", "Could not find your registration. Please contact support.");

        // 2. Verify payment with Paystack (don't trust the callback alone)
        // The webhook handler will also process this, but we verify here for UX
        var tenant = subscription.Tenant;

        if (tenant.Status == TenantStatus.PendingSetup)
        {
            // Provision the tenant database
            var provisionResult = await _provisioner.ProvisionAsync(new TenantProvisionRequest(
                tenant.Name, tenant.Slug, tenant.ContactEmail, tenant.PlanId
            ));

            if (!provisionResult.Success)
                return SwapView("Error", "Account setup is in progress. You'll receive an email shortly.");
        }

        return SwapView("Success", new RegisterSuccessViewModel
        {
            TenantSlug = tenant.Slug,
            TenantName = tenant.Name
        });
    }
}
```

## 8. Invoice Generation

Invoices are generated automatically when payments are processed:

```csharp
public class InvoiceGenerator
{
    private readonly CoreDbContext _coreDb;

    public async Task<Invoice> GenerateAsync(Guid tenantId, Guid subscriptionId,
        decimal amount, string currency)
    {
        // Generate sequential invoice number: INV-{YEAR}-{SEQUENCE}
        var year = DateTime.UtcNow.Year;
        var lastInvoice = await _coreDb.Invoices
            .Where(i => i.InvoiceNumber.StartsWith($"INV-{year}-"))
            .OrderByDescending(i => i.InvoiceNumber)
            .FirstOrDefaultAsync();

        var sequence = 1;
        if (lastInvoice is not null)
        {
            var lastNum = lastInvoice.InvoiceNumber.Split('-').Last();
            if (int.TryParse(lastNum, out var parsed))
                sequence = parsed + 1;
        }

        var invoice = new Invoice
        {
            TenantId = tenantId,
            SubscriptionId = subscriptionId,
            InvoiceNumber = $"INV-{year}-{sequence:D4}",
            Amount = amount,
            Currency = currency,
            Status = InvoiceStatus.Issued,
            IssuedDate = DateTime.UtcNow,
            DueDate = DateTime.UtcNow, // Due immediately for auto-billing
        };

        _coreDb.Invoices.Add(invoice);
        await _coreDb.SaveChangesAsync();

        return invoice;
    }
}
```

## 10. Plan Sync with Paystack

On application startup (production only), plans are synced to Paystack:

```csharp
public class PaystackPlanSyncService : BackgroundService
{
    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        using var scope = _scopeFactory.CreateScope();
        var billing = scope.ServiceProvider.GetRequiredService<IBillingService>();
        await billing.SyncPlansAsync();
    }
}
```

The `SyncPlansAsync` method in `PaystackBillingService`:
1. Lists all plans from Paystack API
2. Compares with plans in `CoreDbContext`
3. Creates missing plans in Paystack
